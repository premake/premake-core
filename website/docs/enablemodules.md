Sets whether or not the compiler should enable C++20 modules.

```lua
enablemodules ("value")
```

### Parameters ###

`value` is one of:

| Value | Description |
|-------|-------------|
| On    | Enable module support |
| Off   | Disable module support |

### Applies To ###

Project configurations.

### Availability ###

Premake 5.0.0-beta3 or later for Visual Studio 2019 and later.
