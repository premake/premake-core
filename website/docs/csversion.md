Specifies the C# language level.

```lua
csversion ("value")
```

### Parameters ###

`value` is a string specifying the C# language level.

### Applies To ###

Project configurations.

### Availability ###

Premake 5.0.0-alpha1 or later.

### Examples ###

```lua
csversion "6"
```
