Specifies the remote debug port.

```lua
debugport (portnumber)
```

### Parameters ###

`portnumber` is an integer port number for the debugger to connect on.

### Applies To ###

Project configurations.

### Availability ###

Premake 5.0.0-alpha1 or later for CodeLite.

### Examples ###

```lua
debugport 1337
```
