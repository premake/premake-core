Excludes a project from the build.

```lua
excludefrombuild "On"
```

### Parameters ###

*value* specifies whether to exclude project from build:

| Value       | Description                                  |
|-------------|----------------------------------------------|
| On          | Excludes a project from the build.           |
| Off         | Default behavior, includes project in build. |

### Applies To ###

Project configurations.

### Availability ###

Premake 5.0-beta8 or later on Visual Studio.
