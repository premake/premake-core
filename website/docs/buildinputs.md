Specifies the source file file inputs of a custom build command or rule.

```lua
buildinputs { "inputs" }
```

### Parameters ###

`inputs` is the list of input source files.


### Applies To ###

Project configurations.


### Availability ###

Premake 5.0.0-alpha1 or later.
