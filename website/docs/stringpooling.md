Specifies if string pooling should be used.

```lua
stringpooling ("value")
```

### Parameters ###

`value` is one of:

| Value | Description |
|-------|-------------|
| On | Enable string pooling. |
| Off | Disable string pooling. |

### Applies To ###

Project configurations.

### Availability ###

Premake 5.0.0-alpha12 or later for Visual Studio or `msc` toolset.

### Examples ###

```lua
stringpooling "Off"
```
