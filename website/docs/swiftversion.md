Specifies the version of Swift to compile with.

```lua
swiftversion ("value")
```

### Parameters ###

`value` is one of:

| Value | Description |
|-------|-------------|
| 4.0 | Swift Language Version 4.0 |
| 4.2 | Swift Language Version 4.2 |
| 5.0 | Swift Language Version 5.0 |

## Applies To ###

Project configurations.

### Availability ###

Premake 5.0.0-beta1 or later.

### Examples ###

```lua
swiftversion "4.2"
```
