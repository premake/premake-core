Generates assembly-level debug information in Visual Studio projects

```lua
assemblydebug(enable)
```

### Parameters ###

`enable` **boolean** - enable/disable assemblydebug.

## Applies To ###

The `config` scope.

### Availability ###

Premake 5.0.0 alpha 16 or later.

### Examples ###

```lua
assemblydebug(true)
```
